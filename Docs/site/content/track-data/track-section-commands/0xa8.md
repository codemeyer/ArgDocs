# 0xA8 - Section That Triggers the Chequered Flag (Track Section Command)

This command defines which track section that triggers the chequered flag man to be visible.
When the leader of the race drives into the section on the final lap, the flag man will appear
at the start/finish line.

The command always occurs just _once_ per track.
