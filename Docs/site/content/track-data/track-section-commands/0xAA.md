# 0xAA - Computer cars pit lane modifier? (Track Section Command)

This command is never used more than _once_ for a track.

If it is used, it is always declared in the first track section.


## Arguments

<table class="table table-bordered table-striped table--medium">
    <thead>
        <tr>
            <th>Arg</th>
            <th>Description</th>
            <th>Common values</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>0</td>
            <td>Unknown/unused?</td>
            <td>Always 0</td>
        </tr>
        <tr>
            <td>1</td>
            <td>Length for computer cars to angle towards pitlane (backwards)?</td>
            <td>0 to 70</td>
        </tr>
        <tr>
            <td>2</td>
            <td>Length for computer cars to angle out of pits?</td>
            <td>0, 7 or 60</td>
        </tr>
        <tr>
            <td>3</td>
            <td>Computer car pit lane speed</td>
            <td>6656, 11264 or 12800</td>
        </tr>
    </tbody>
</table>
