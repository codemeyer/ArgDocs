# 0x9B - Unknown Pit Lane Marker 1 (Track Section Command)

This command always occurs _once_ per track, in the first section of the pit lane.

More information about the pit lane specific commands and how they are placed
can be found [here](/argdocs/track-data/pit-lane-commands/).
