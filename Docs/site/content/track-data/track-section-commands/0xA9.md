# 0xA9 - Pit Lane View Distance (Track Section Command)

Each track also has a default pit lane view distance defined elsewhere.


## Arguments

<table class="table table-bordered table-striped table--medium">
    <thead>
        <tr>
            <th>Arg</th>
            <th>Description</th>
            <th>Common values</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>0</td>
            <td>Unknown/unused?</td>
            <td>Always 0</td>
        </tr>
        <tr>
            <td>1</td>
            <td>Distance in which the pitlane can be seen?</td>
            <td>0, 1, 3, 4, 6 or 29</td>
        </tr>
    </tbody>
</table>
