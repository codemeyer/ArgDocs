# Track-side Flat Objects

These are "flat" objects that are placed track-side, such as trees and bushes.

They are usually placed with ID 5. Some of these objects (e.g. road signs) are also placed using ID 1 or 13.

Images coming in due time.

Interesting to note is that these IDs were carried over to Grand Prix 2 on a one-to-one basis, see e.g. the GP2 [ID5 Reference Card](http://www.grandprix2.de/Anleitung/tutus/id5/ID5.html)
